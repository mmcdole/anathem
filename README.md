# Anathem

`Anathem` is a self-hosted digital library for your ebooks and audiobooks.

#### Packages:

Package | Description
--- | ---
`api` | REST API
`cmd` | CLI utility 
`config` | Application Settings
`database` | Datastore 
`importer` | Book Importers
`metadata` | Metadata Providers
`web` | Web front-end
